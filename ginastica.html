Análise de Dados da Final de Ginástica Artística - Paris 2024

Visão Geral do Projeto

Este projeto consiste em uma análise detalhada dos dados da final de ginástica artística dos Jogos Olímpicos de Paris 2024. O objetivo foi explorar o desempenho dos 24 atletas finalistas, utilizando técnicas de análise estatística e visualização de dados para extrair insights sobre as pontuações em diferentes aparelhos.

A base de dados foi coletada a partir do relatório oficial dos Jogos Olímpicos de Paris 2024, garantindo a precisão e relevância das informações analisadas.

Tecnologias Utilizadas

Python Jupyter Notebook Pandas Plotly Análise Estatística

Principais Análises e Visualizações

O núcleo da análise foi conduzido em um Jupyter Notebook, onde diversas técnicas foram aplicadas:

  • Cálculo de estatísticas descritivas como média, mediana e moda das pontuações por aparelho.
  • Geração de boxplots para visualizar a distribuição das notas e identificar outliers em cada aparelho.
  • Criação de histogramas para entender a frequência das pontuações.
  • Análise comparativa do desempenho entre os atletas e aparelhos.
